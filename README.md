## EDUTUNE: A STACKED ENSEMBLE APPROACH TO PREDICTING ENGAGEMENT AND PERFORMANCE IN EDUCATION.
### Project Overview:

EDUTUNE is an AI-driven educational analytics platform designed to enhance student learning and academic performance through data-driven insights. It combines machine learning, behavioral analytics, and dashboard visualizations to evaluate engagement, predict student outcomes, and personalize learning paths.

### Objectives:
> Analyze students’ learning behavior and engagement metrics.

> Predict academic performance using ML algorithms.

> Generate personalized learning recommendations.

> Assist educators through performance dashboards.

> Promote adaptive, data-based learning systems.

### Technical Stack:
Category	Tools / Frameworks

Programming	Python

ML Libraries	Scikit-learn, Pandas, NumPy

Visualization	Matplotlib, Seaborn, Power BI

Modeling	Linear Regression, Random Forest, Hybrid Stacking Regressor

Documentation	MS Word, LaTeX

Environment	Jupyter Notebook / VS Code
### Core Methodology:
1. Data Collection – Student datasets with performance and engagement indicators.

2. Preprocessing – Cleaning, normalization, and feature selection.

3. Model Development – Linear Regression, Random Forest, and Hybrid Stacking Regressor.

4. Evaluation – Using metrics such as MAE, MSE, R².

5. Visualization – Interactive dashboards to display insights.

6. Recommendation Engine – Suggests personalized learning interventions.

### Key Results:
> The Hybrid Stacking Regressor outperformed other baseline models.

> Improved prediction accuracy for student performance and engagement.

> Strong correlation between study habits and outcome metrics.

> Provided actionable insights for teachers and students.
